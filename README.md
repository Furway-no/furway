## Development

### How to Run Locally

Prerequisites:

- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/install/)

Run attached to your terminal (so you wont forget to stop the container when you are done):

```bash
docker-compose up
```
